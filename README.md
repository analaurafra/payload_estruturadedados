# Payload Document Samples

This repository contains sample payloads, JSON data files, and HTML templates used for document generation with Brazilian financial and legal scenarios.

## Contents

- `AberturaConta/`
  - Payload examples for account opening processes.
  - Includes a JavaScript payload file and an HTML template for generating account opening documents.

- `AlienacaoFiduciara/`
  - Payload and template files for fiduciary alienation agreements.
  - Includes JSON data and an HTML model document.
  - Includes an example output HTML file.

- `AlienacaoFiduciariaComplexa01/`
  - A more complex fiduciary alienation payload example.
  - Includes JSON payload data and an HTML template representing a complex agreement.
  - Includes a sample generated output HTML file.

- `AlienacaoFiduciariaComplexa02/`
  - Additional fiduciary alienation examples and payload structures.

## Purpose

These documents are intended to support development and testing of document generation workflows, such as populating contract templates with structured payload data.

## Notes

- File names are primarily in Portuguese because the examples target Brazilian document formats.
- Use the JSON and JS payload files to populate the corresponding HTML templates when generating final documents.
- Output HTML files show example rendered results for the payload/template combinations.
- The following tools are used for editing and validating payloads and templates:
  - https://jsonlint.com/
  - https://jsoncrack.com/editor
  - https://handlebarsjs.com/examples
